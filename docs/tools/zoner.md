Draw zones (polygons and linepoints) on maps using markers.

**URL:** [https://zoner.onlyworlds.com](https://zoner.onlyworlds.com)
**Type:** Web Application

---

## Features

- **Zone Drawing** - Click to place markers, create polygons or lines
- **Live Editing** - Drag markers, insert points, delete with right-click
- **Map Support** - Image URL input or work on grid fallback
- **API Integration** - Optional sync to OnlyWorlds as Map, Zone, Marker elements
- **Local Workflow** - Work offline, connect when ready
- **Export** - JSON format compatible with Base Tool migrator
