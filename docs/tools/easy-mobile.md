Quick mobile web editor for OnlyWorlds elements.

**URL:** [mobile-crud-base.pages.dev](https://easy-mobile.onlyworlds.com) 
**Type:** Mobile Web Application

---

## Features

- Full CRUD - All 22 element types with inline editing
- Dual-Mode - Online (API) or offline (local IndexedDB) with sync
- Mobile-first - Touch-friendly, compact layout 
- Search & Browse - Global search, infinite scroll, recent elements
- Collapsible sections - Field grouping by semantic purpose
- Import/Export - Load from JSON, paste from clipboard, export world
