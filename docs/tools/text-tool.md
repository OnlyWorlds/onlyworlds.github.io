Multi-purpose tinker tool for AI parsing, YAML editing, and earning token ratings.

**URL:** [onlyworlds.com/text_tool](https://onlyworlds.com/text_tool)
**Type:** Web Application

---

## Features

- **AI Extraction** - Parse text into structured world data (uses tokens)
- **YAML Editing** - Direct editing of element data
- **Conversational Refinement** - Chat-based element improvement
- **Token Rating** - Earn rating through feedback submissions
- **Bot Import** - Import from [OnlyWorldsBot](https://chatgpt.com/g/g-dydgDFnOz-OnlyWorldsBot) without token costs
