# Tool history

A few tools have come and gone. They served their purpose, and their work carries on elsewhere.

- **text tool** — an AI-parsing and YAML-editing tinker tool, retired with the 2026 platform rebuild. Parsing now lives in the [toolkit](https://github.com/OnlyWorlds/toolkit) and the [MCP server](https://www.onlyworlds.com/mcp).
- **map tool** — maps, pins, and hierarchies, retired with the same rebuild. Mapping now lives in [Atlas](https://atlas.onlyworlds.com).
- **the old browser editor** — the first web workspace, superseded by [Atlas](https://atlas.onlyworlds.com) and the [onlyworlds.com](https://www.onlyworlds.com) portal.
