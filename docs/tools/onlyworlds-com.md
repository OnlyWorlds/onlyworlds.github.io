The OnlyWorlds platform hub: accounts, world hosting, keys, and the API. Editing happens in [Atlas](atlas) and the other tools; onlyworlds.com is where your worlds live online.

**URL:** [onlyworlds.com/account](https://www.onlyworlds.com/account/)
**Type:** Web Application

---

## Features

- **World hosting** - Create worlds and manage them from the account portal
- **Keys** - Mint and revoke scoped world keys (`ow_w_` write / `ow_r_` read, shown once)
- **Account tokens** - Mint `ow_a_` tokens so apps can act on your account (Settings)
- **Element viewer** - Browse any world's elements read-only, straight from the portal
- **Export** - Download a world as a single JSON file
- **API** - Full REST access at [/api/docs](https://www.onlyworlds.com/api/docs) and an [MCP server](https://www.onlyworlds.com/mcp) for AI assistants

## Getting Started

1. Create an account at [onlyworlds.com](https://www.onlyworlds.com/accounts/login/) (email or Google/GitHub/Discord)
2. Create a world - your first world sets your 4-digit PIN, the write wall on all your worlds
3. Mint a world key from the world page for tool and API access
4. Open the world in [Atlas](https://atlas.onlyworlds.com) or any other tool
